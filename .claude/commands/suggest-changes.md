# Suggest Lyric Changes

Suggest modifications to make lyrics more singable while preserving meaning.

## Instructions

Review the lyrics and suggest changes for:

1. **Syllable Consistency**
   - Identify lines with irregular syllable counts
   - Suggest word substitutions to normalize

2. **Stress Alignment**
   - Find words where natural stress conflicts with musical stress
   - Suggest alternatives that flow better

3. **Breath Points**
   - Ensure natural pauses for breathing
   - Avoid long phrases without breaks

4. **Vowel Sounds**
   - Prefer open vowels on sustained notes (ah, oh, ee)
   - Avoid consonant clusters on held notes

5. **Rhyme Enhancement**
   - Strengthen weak rhymes if needed
   - Maintain original rhyme scheme

## Output Format

For each suggested change:

```
### Line X: "[original line]"

**Issue**: [description of problem]

**Suggestion**: "[modified line]"

**Rationale**: [why this improves singability]

**Original meaning preserved**: Yes/Partially/No (explain if partial)
```
