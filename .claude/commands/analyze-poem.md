# Analyze Poem

Analyze the provided poem for musicality and singability.

## Instructions

When analyzing a poem, examine:

1. **Structure**
   - Number of stanzas
   - Lines per stanza
   - Overall form (sonnet, free verse, ballad, etc.)

2. **Meter**
   - Count syllables per line
   - Identify stress patterns (iamb, trochee, anapest, dactyl)
   - Note any irregularities

3. **Rhyme**
   - Identify rhyme scheme (ABAB, AABB, etc.)
   - Note slant rhymes and internal rhymes
   - Mark unrhymed lines

4. **Singability Assessment**
   - Flag lines with awkward syllable counts
   - Identify words difficult to sustain (clusters of consonants)
   - Note natural breathing points
   - Assess emotional arc for melody mapping

## Output Format

```
## Poem Analysis

### Structure
- Stanzas: X
- Lines per stanza: X, X, X
- Form: [type]

### Meter Analysis
Line 1: [syllables] - [pattern] - [notes]
Line 2: ...

### Rhyme Scheme
[ABAB or similar notation]

### Singability Notes
- Strengths: ...
- Challenges: ...
- Suggested focus areas: ...
```
